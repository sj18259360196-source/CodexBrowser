# Codex Browser Edge Relay

1. Open `edge://extensions` in the user's ordinary Microsoft Edge profile.
2. Enable **Developer mode**, choose **Load unpacked**, and select this directory.
3. In Codex Browser's runtime settings, open the two-minute pairing window.
4. Click the extension icon and choose **Connect local Codex Browser**.
5. Select `edge-extension` as the preferred runtime and restart Codex Browser.

The extension uses the user-visible `debugger` permission. It never reads or exports Cookie values or saved passwords. CAPTCHA, Turnstile, MFA, passkeys, and sensitive fields remain manual-only.
